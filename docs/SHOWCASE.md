# Visual Showcase

This document provides a visual walkthrough of the `rajathkiran.io` portfolio, designed to help recruiters and contributors quickly understand the user interface without running the code.

---

## 🏠 Homepage (About Section)

The entry point of the application, featuring a high-level summary and the identity sidebar.

![Homepage Placeholder](docs/images/homepage.png)

- **Description**: A clean, dark-themed dashboard overview.
- **Key Functionality**: Three.js interactive background, sticky sidebar, and smooth scrollable main content area.
- **Recruiter Value**: Immediate visibility of name, role, location, and core capabilities.

---

## 🗂️ Sidebar & Identity

The persistent left-hand navigation column.

![Sidebar Placeholder](docs/images/sidebar.png)

- **Description**: A glassmorphic card containing the developer's avatar and metadata.
- **Key Functionality**: Expanding contact chips, social media icon links.
- **Recruiter Value**: One-click access to Email, LinkedIn, GitHub, and location data at any point during navigation.

---

## 📜 Resume & Experience

The detailed professional history section.

![Resume Placeholder](docs/images/resume.png)

- **Description**: A vertical timeline representing chronological experience.
- **Key Functionality**: CSS-animated timeline dots and a dynamic "Skills Radar" with animated progress bars.
- **Recruiter Value**: Easy-to-read, structured breakdown of where the developer has worked and what technologies they have mastered.

---

## 🚀 Projects Grid

The technical portfolio showcase.

![Projects Placeholder](docs/images/projects.png)

- **Description**: A responsive grid of project cards with hover-reveal overlays.
- **Key Functionality**: Category filtering (Web, AI, Cloud), Recruiter Mode optimization, and modal triggers.
- **Recruiter Value**: Quickly filters out noise to focus exactly on the domain the recruiter is hiring for.

---

## 🎓 Certificates Viewer

A dedicated space for verified credentials.

![Certificates Placeholder](docs/images/certificates.png)

- **Description**: A gallery of thumbnails representing earned certifications.
- **Key Functionality**: Integrated modal system that pops up full-resolution PDFs/images without leaving the page.
- **Recruiter Value**: Instantly proves technical claims made in the resume section.

---

## 🤖 AI Chat Assistant

The interactive resume query tool.

![AI Chat Placeholder](docs/images/ai-chat.png)

- **Description**: A floating chat widget docked to the bottom right of the screen.
- **Key Functionality**: Real-time communication with a fine-tuned Gemini AI model.
- **Recruiter Value**: Allows recruiters to ask specific, conversational questions (e.g., "Does Rajath have GCP experience?") and get immediate, contextual answers.
