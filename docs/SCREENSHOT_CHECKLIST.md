# Screenshot Capture Checklist

To complete the visual documentation of this repository (specifically the `README.md` and `SHOWCASE.md` files), the following screenshots must be captured and placed in the `docs/images/` directory.

### Requirements
- **Format**: PNG or WebP
- **Resolution (Desktop)**: 1920x1080 (or similar 16:9 ratio)
- **Resolution (Mobile)**: 390x844 (iPhone 12/13/14 Pro dimensions)
- **Theme**: Ensure Dark Theme / Default Theme is active

---

- [ ] `homepage-desktop.png` - The main About page, capturing the hero intro and timeline.
