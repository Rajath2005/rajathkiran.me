MediQ Health Platform

What is MediQ Health?

MediQ Health is a comprehensive Ayurvedic consultation platform that combines modern web technologies with traditional health practices. It provides users with AI-driven treatment recommendations, real-time consultation tracking, and personalized wellness plans tailored to their specific symptoms and health profiles.

Problem Statement

Managing continuous Ayurvedic treatments involves tracking complex daily regimens, diet plans, and herbal prescriptions. Existing healthcare platforms often cater strictly to allopathic models and lack the nuanced data structures needed to record Dosha types and holistic lifestyle recommendations effectively.

Tech Stack

  • Frontend: React.js, Tailwind CSS
  • Backend: Node.js, Express
  • Database & Authentication: Supabase (PostgreSQL)
  • Hosting: Netlify

Architecture & Implementation

MediQ was engineered with a focus on seamless user experience and strict data security for health records.

The backend utilizes Supabase for rapid API generation and secure row-level security (RLS) policies, ensuring that users can only access their own health data. The React frontend consumes these APIs to provide a dashboard where users can track their ongoing treatment plans, view historical consultation notes, and receive automated, AI-driven wellness nudges based on their tracked symptoms.

Results & Impact

The platform delivers a highly responsive, app-like experience on both desktop and mobile devices. By integrating secure backend architecture with a modern UI, MediQ successfully reduces the friction of managing holistic treatment regimens and provides a foundation for the scalable integration of AI health models (such as those later expanded in AyuDost).
